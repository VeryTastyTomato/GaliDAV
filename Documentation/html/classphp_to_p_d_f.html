Parameters:
$w- width $h - line height $txt - the outputed text $border - border(LRTB 0 or 1) $align - horizontal align 'JLR' $fill - fill (1/0) $vh - vertical adjustment - the Multicell Height will be with this VH Higher!!!! $valign - Vertical Alignment - Top, Middle, Bottom
